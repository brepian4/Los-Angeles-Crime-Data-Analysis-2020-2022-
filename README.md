# Los Angeles Crime Data Analysis (2020–2022)

## Key Findings
- The location with the **highest crime frequency** is **800 N Alameda Street**, which corresponds to **LA Union Station**.
- This location recorded **768 crime incidents** between **2020 and 2022**.
- The geographical coordinates of this hotspot are:
  - **Latitude:** 34.0561  
  - **Longitude:** -118.2375
- The identified location was **verified using Google Earth** to ensure geographical accuracy.
- The area falls under the **Central LA** region.
- These findings can support **urban safety planning**, **law enforcement resource allocation**, and **crime prevention strategies**.
